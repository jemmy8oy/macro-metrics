# F4.1 — Indicator Card Component

**Epic:** [E4 — Standalone Indicator Cards](../epics/e4-indicator-cards.md)

## Summary

A reusable card for metrics that are already ratios or rates and cannot be used in a comparison (CAPE, bond yields). Shows the current value, a sparkline, and a deviation label.

## What the user sees

- Large headline number (current value)
- Deviation label: e.g. *"90% above historical average (17×)"*
- Sparkline of the last 30 years
- No date range selector needed

## UI/UX questions to resolve in Phase 2
- Same card dimensions as ratio cards, or a distinct narrower format?
- Should the sparkline show the long-run average as a reference line too?
