# F4.1 — Indicator Card Component

**Epic:** [E4 — Standalone Indicator Cards](../epics/e4-indicator-cards.md)

## Summary

A reusable card component for standalone indicators (CAPE, bond yields). Shows a large headline value, a sparkline of historical data, and a deviation label relative to the long-run average.

## Props

```ts
interface IndicatorCardProps {
  label: string;
  unit: string;
  series: { date: string; value: number }[];
  longRunAverage: number;
}
```

## Layout

```
┌──────────────────────────┐
│  Shiller CAPE Ratio      │
│  32.4×                   │  ← current value (large)
│  90% above avg (17×)     │  ← deviation label
│  ▁▂▃▄▅▆▇█ (sparkline)   │  ← last 30 years
└──────────────────────────┘
```
