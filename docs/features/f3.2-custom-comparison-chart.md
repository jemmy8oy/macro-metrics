# F3.2 — Custom Comparison Chart

**Epic:** [E3 — Custom Comparison Tool](../epics/e3-custom-comparison.md)

## Summary

The ratio chart rendered after the user picks a numerator and denominator. Reuses the shared `RatioChart` component (F2.2). Label is auto-generated from the two metric names.

## Behaviour

- Chart only appears once both metrics are selected
- Shows a loading state while data is fetching
- Shows an inline error if the fetch fails
- Date range selector shared with the same component used in preset cards

## UI/UX questions to resolve in Phase 2
- Should the custom comparison chart be full-width, or the same size as a preset card?
- Is there a distinct visual separation between the preset section and the custom section?
