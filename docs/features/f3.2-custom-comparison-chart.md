# F3.2 — Custom Comparison Chart

**Epic:** [E3 — Custom Comparison Tool](../epics/e3-custom-comparison.md)

## Summary

The chart section rendered after the user selects a numerator and denominator via the metric picker (F3.1). Reuses the shared `RatioChart` component with data fetched from `GET /api/metrics/ratio`.

## Behaviour

- Renders only once both numerator and denominator are selected
- Fetches from `/api/metrics/ratio?numerator={id}&denominator={id}`
- Shows loading skeleton while fetching
- Shows an inline error state if the API call fails
- Chart label auto-generated: e.g. *"Gold / S&P 500"*
- Date range selector defaults to full available history

## State management

Numerator and denominator selections held in local component state (no Redux needed — ephemeral UI state).
