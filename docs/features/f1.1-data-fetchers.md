# F1.1 — External Data Fetchers

**Epic:** [E1 — Data Ingestion & API](../epics/e1-data-ingestion.md)

## Summary

Three fetcher services, one per provider, each responsible for fetching raw time-series data and returning it in a normalised internal format.

## Fetchers

### yfinance fetcher
Covers: Gold (`GC=F`), Oil (`BZ=F`), FTSE 100 (`^FTSE`), S&P 500 (`^GSPC`), Bitcoin (`BTC-USD`)
- Use `yfinance` Python library via a sidecar script or a .NET HTTP call to a lightweight Python FastAPI microservice, OR use a .NET yfinance-compatible library
- Returns monthly OHLC — use adjusted close

### FRED fetcher
Covers: US CPI (`CPIAUCSL`), US Wages (`CES0500000003`), US House Prices (`CSUSHPISA`), CAPE (`CAPE`), US 10yr Yield (`DGS10`)
- FRED REST API: `https://api.stlouisfed.org/fred/series/observations`
- Requires a free API key (stored in `appsettings`)
- Returns monthly observations

### ONS fetcher
Covers: UK CPI, UK Wages (EARN01), UK House Prices (HPI)
- ONS Beta API: `https://api.ons.gov.uk/`
- No API key required
- Returns quarterly or monthly time series depending on dataset

## Internal model

All fetchers return:
```csharp
IEnumerable<DataPoint>
// DataPoint: { Date, Value }
```
