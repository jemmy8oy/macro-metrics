# F4.2 — Indicators Section

**Epic:** [E4 — Standalone Indicator Cards](../epics/e4-indicator-cards.md)

## Summary

The bottom section of the homepage displaying all three standalone indicator cards in a row. Each card fetches its own data via `GET /api/metrics/{id}`.

## Indicators displayed

| Metric ID | Label |
|---|---|
| `cape` | Shiller CAPE Ratio |
| `uk-10yr-gilt` | UK 10yr Gilt Yield |
| `us-10yr-treasury` | US 10yr Treasury Yield |

## Layout

Three `IndicatorCard` components in a responsive row (stack to single column on mobile — post-MVP).
