# F2.2 — Ratio Chart Component

**Epic:** [E2 — Dashboard & Preset Ratios](../epics/e2-dashboard-presets.md)

## Summary

A reusable chart component shared by preset cards (E2) and the custom comparison tool (E3). Renders a ratio time series as a line chart with a long-run average reference line and deviation tooltip.

## Chart spec

- **Library:** Recharts (lightweight, composable, works well with RTK Query data)
- **X-axis:** Date (monthly)
- **Y-axis:** Ratio value
- **Series line:** ratio over time
- **Reference line:** long-run average (dashed, labelled)
- **Tooltip:** date, current value, % above/below long-run average
- **Colour:** ratio line warm (above avg) / cool (below avg) — or single colour with threshold annotation
- **Date range selector:** optional `from`/`to` inputs; defaults to full history

## Props

```ts
interface RatioChartProps {
  label: string;
  series: { date: string; value: number }[];
  longRunAverage: number;
  from?: string;
  to?: string;
}
```
