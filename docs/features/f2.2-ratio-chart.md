# F2.2 — Ratio Chart Component

**Epic:** [E2 — Dashboard & Preset Ratios](../epics/e2-dashboard-presets.md)

## Summary

A reusable chart used by both the preset cards (E2) and the custom comparison tool (E3). Renders a ratio time series with a long-run average reference line.

## What the user sees

- A line chart of the ratio value over time
- A dashed reference line at the long-run average
- A tooltip showing: date, current ratio value, and % above/below the long-run average
- A date range selector (default: full available history)
- Subtle colour shift when ratio is above vs below the average

## UI/UX questions to resolve in Phase 2
- How compact should the card-sized version of this chart be vs the full-width custom comparison version?
- Should the date range selector be a slider, dropdown presets (5y / 10y / 20y / Max), or free date inputs?
- Warm/cool colour scheme — or a single colour with a threshold annotation line?
