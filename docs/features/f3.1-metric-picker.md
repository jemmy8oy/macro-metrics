# F3.1 — Metric Picker UI

**Epic:** [E3 — Custom Comparison Tool](../epics/e3-custom-comparison.md)

## Summary

Two dropdowns that let the user select any two comparison-eligible metrics as numerator and denominator. Drives the custom comparison chart in real time.

## Behaviour

- Populated from `GET /api/metrics`, filtered to `isIndicatorOnly: false`
- Numerator and denominator cannot be the same — the opposing dropdown disables the currently selected option
- On change, the custom comparison chart re-fetches immediately
- Default state: no selection (chart shows empty/placeholder)

## Component

```
[UK House Prices  ▼]  /  [UK Wages  ▼]
```

- Labels from the metric catalogue
- Controlled component — selection stored in local React state (or Redux slice)
