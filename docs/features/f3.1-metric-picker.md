# F3.1 — Metric Picker UI

**Epic:** [E3 — Custom Comparison Tool](../epics/e3-custom-comparison.md)

## Summary

Two dropdowns that let the user select any two comparable metrics as numerator and denominator. On selection the chart updates immediately.

## Behaviour

- Numerator and denominator cannot be the same metric
- Standalone indicators (CAPE, bond yields) are excluded from the picker
- Chart renders once both selections are made
- Selections are ephemeral — not persisted between sessions for MVP
