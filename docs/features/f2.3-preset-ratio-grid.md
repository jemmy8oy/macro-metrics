# F2.3 — Preset Ratio Card Grid

**Epic:** [E2 — Dashboard & Preset Ratios](../epics/e2-dashboard-presets.md)

## Summary

The preset ratio cards displayed on the homepage. Each card is a self-contained unit that fetches its own data via the `/api/metrics/ratio` endpoint and renders using the shared `RatioChart` component.

## Preset definitions

| # | Title | Numerator | Denominator |
|---|---|---|---|
| 1 | UK Affordability | `uk-house-prices` | `uk-wages` |
| 2 | US Affordability | `us-house-prices` | `us-wages` |
| 3 | UK Real House Prices | `uk-house-prices` | `uk-cpi` |
| 4 | Real Gold | `gold` | `us-cpi` |
| 5 | Gold vs Equities | `gold` | `sp500` |
| 6 | Real S&P 500 | `sp500` | `us-cpi` |
| 7 | UK Property vs Gold | `uk-house-prices` | `gold` |
| 8 | Real Oil | `oil` | `us-cpi` |
| 9 | Bitcoin vs Gold | `bitcoin` | `gold` |

These are defined as a static config array in the frontend — no backend changes needed to add/remove presets.

## Card layout

Each card shows:
- Title
- Current ratio value + % above/below average (headline stat)
- `RatioChart` (compact height ~200px)
