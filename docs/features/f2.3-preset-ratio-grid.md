# F2.3 — Preset Ratio Card Grid

**Epic:** [E2 — Dashboard & Preset Ratios](../epics/e2-dashboard-presets.md)

## Summary

The homepage displays 9 curated ratio cards, each showing a meaningful macro comparison at a glance. The preset list is static config — no backend changes needed to add or remove cards.

- Should cards be expandable/clickable to show a full-size chart?
- Ordering — group by theme (affordability, inflation-adjusted, cross-asset) or flat list?
